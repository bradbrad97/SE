### Functional

- 상품정보조회

input: 검색조건

output: 상품명, 구매만족도 출력

- 정렬하여 출력

input: 요청

output: 평균 구매 만족도 출력

- 상품 상세 정보 출력

input: 상품 선택

output: 상품 상세 정보 출력

- 구매

input:  요청

output: 구매처리

- 포인트 사용

input: 포인트 사용 요청

output: 가격에서 사용 포인트만큼 할인, 포인트 차감

- 포인트 적립

input: 구매

output: 금액의 1% 적립

- 포인트 반환

input: 환불

output: 사용한 포인트 반환

- 추가 상품 구매

input: 추가 상품 선택

output: 추가 상품 구매처리 및 총 구매금액 연산
